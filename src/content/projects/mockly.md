An AI-powered mock interview platform that simulates real interview scenarios. Built using the Vapi SDK for voice interaction, Gemini for AI-driven question generation and feedback, and Firebase for real-time data storage and user authentication. GitHub Actions are used to automate deployment and performance monitoring.
